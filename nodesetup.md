# Node js setup in vps server

---

## node js and install in vps server

### install node js

```bash
sudo apt install nodejs
```

## check nodejs installation successs ?

```bash
nodejs --version
```

- [x] nodejs install

---

### install npm js

```bash
sudo apt install -g  npm
```

## check npm installation successs ?

```bash
npm --version
```

- [x] npm install
